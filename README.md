# Programacion PLC, Neumática e Hidraúlica

Recopilación de proyectos desarrollados en programación de PLC aplicados a la automatización industrial. Incluye ejemplos de control de procesos, secuencias automáticas, manejo de entradas y salidas digitales y analógicas, así como la integración con sensores, actuadores y sistemas HMI.

Los programas fueron diseñados utilizando lenguajes estándar de automatización como Ladder Diagram (LD) y se enfocan en optimizar la eficiencia, seguridad y confiabilidad de los sistemas industriales.
